# 機能

## Create

入力を受け取る

ファイルに一行追加する

## Read

ファイルをすべて読み取る

一行ずつフォーマットして表示する。

## Update

入力を受け取る

ID と対応する行を取得する。

取得した行データを更新する。

対応行を削除する。

更新データ行を追記する。

## Delete

入力を受け取る

ID と対応する行を取得する。

取得できた場合、ID と対応する行を削除する
